1. テトリス基本仕様書 (Geminiへのインプット用)
以下の内容をコピーしてGeminiに渡すと、共通認識を持って開発をスタートできます。

【ゲーム概要】
タイトル: シンプル・テトリス

プラットフォーム: Webブラウザ (HTML5 / Canvas / JavaScript)

開発言語: HTML, CSS, JavaScript (Vanilla JS)

【システム仕様】
フィールド: 横10 × 縦20 マス

テトリミノ: 7種類（I, O, T, S, Z, J, L）

各ミノに固有の色を割り当てる

操作方法:

← →: 左右移動

↓: ソフトドロップ（加速）

↑: 回転（右回転）

Space: ハードドロップ（一気に接地）

基本ロジック:

自由落下（レベルに応じた速度）

衝突判定（壁、床、他のブロック）

ライン消去（1列揃ったら消去し、上のブロックを下げる）

ゲームオーバー判定（出現位置がブロックで埋まっている場合）

2. 段階的開発ステップ（プロンプト例）
Geminiとの対話で、以下の順に指示を出してください。

ステップ1：土台作り
プロンプト:
「テトリスをHTML/CSS/JSで作りたいです。まずは、20x10のグリッドを表示するHTMLとCSS、そしてゲームループの基礎（一定間隔でコンソールに'tick'と出す）JavaScriptコードを作成してください。背景は黒、グリッドの枠線は薄いグレーにしてください。」

ステップ2：ミノの定義と表示
プロンプト:
「次に、テトリミノの形状を定義してください。7種類のミノを2次元配列で表現し、ランダムに1つ選んでフィールドの最上部中央に表示するロジックを追加してください。」

ステップ3：移動と回転
プロンプト:
「キーボード入力（矢印キー）で、現在落下中のミノを左右に動かしたり、回転させたりする機能を追加してください。壁や床にめり込まないよう、衝突判定も実装してください。」

ステップ4：ライン消去とスコア
プロンプト:
「ミノが接地したとき、横一列が揃っていたらその行を消去する機能を追加してください。消去した行数に応じてスコアが加算されるようにし、画面上に表示してください。また、次に出現するミノ（Next）を表示する枠も作ってください。」

3. Geminiを最大限に活かすコツ
「コード実行」をオンにする: GeminiのAdvanced機能（Code Execution）を使っている場合、その場でプレビューを作成してくれることがあります。「実行可能なHTMLファイルを1つにまとめて出力して」と頼むと便利です。

エラーはそのまま貼り付ける: 動かない場合は「〇〇が動きません。コンソールにこのエラーが出ています：[エラー内容]」と送れば、すぐに修正案をくれます。

リファクタリングを頼む: 一通り完成したら「コードが長くなったので、定数、クラス、描画ロジックにファイルを分割するか、整理して」と頼むと、綺麗なコードになります。
